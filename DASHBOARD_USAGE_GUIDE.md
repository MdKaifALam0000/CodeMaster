## Quick Start

### 1. Accessing the Dashboard

There are **3 ways** to open the dashboard:

#### Method 1: Profile Picture Click
1. Click on your profile picture in the top-right navbar
2. A popup will appear with quick options
3. Select "View Profile", "My Progress", or "Settings"

#### Method 2: Dashboard Button
1. Look for the purple "Dashboard" button in the navbar (desktop only)
2. Click it to open the dashboard directly

#### Method 3: Profile Popup
1. Click your profile picture to open the popup
2. Choose any of the menu options to open that specific dashboard section

### üìä Progress Section

**What you'll see:**

#### Summary Cards
- Total problems solved
- Accepted submissions count
- Total submissions count

#### Visual Charts
1. **Difficulty Distribution (Pie Chart)**
   - Shows problems solved by difficulty (Easy/Medium/Hard)
   - Color-coded: Green (Easy), Orange (Medium), Red (Hard)
   - Displays both solved and total counts

2. **Language Performance (Bar Chart)**
   - Compares accepted vs total submissions per language
   - Shows which languages you're most successful with
   - Helps identify areas for improvement

3. **Recent Activity (Line Chart)**
   - Last 14 days of submission activity
   - Blue line: Total submissions
   - Green line: Accepted submissions
   - Track your consistency and progress

#### Progress Bars
- Visual representation of completion for each difficulty
- Shows X/Y format (solved/total)
- Percentage-based progress bars

#### Recent Submissions Table
- Last 10 submissions
- Columns:
  - Problem title
  - Language used
  - Status (Accepted/Wrong/Error)
  - Runtime (milliseconds)
  - Memory usage (MB)
  - Submission date

**Use Cases:**
- Track your learning progress
- Identify weak areas
- Monitor consistency
- Analyze performance trends

### ‚öôÔ∏è Settings Section

**What you can update:**

#### Profile Picture
1. **Upload New Picture:**
   - Click "Upload New Picture" button
   - Select an image file (JPG, PNG, etc.)
   - Max file size: 5MB
   - Image will be automatically cropped to 400x400px
   - Face detection centers your face in the photo
   - Progress bar shows upload status

2. **Delete Picture:**
   - Click "Delete" button next to your current picture
   - Confirm deletion
   - Reverts to initial avatar

#### Personal Information
- **First Name** (Required)
- **Last Name** (Optional)
- **Email** (Read-only, cannot be changed)
- **Age** (6-100 years)

#### Bio
- Write a short description about yourself
- Max 500 characters
- Character counter shows remaining space

#### Skills
- Add your programming skills
- Comma-separated format
- Example: "JavaScript, Python, React, Node.js"
- Displays as colorful badges on profile

#### Social Links
- **GitHub URL**: Your GitHub profile link
- **LinkedIn URL**: Your LinkedIn profile link
- Must be valid URLs

**After Making Changes:**
1. Click "Save Changes" button at the bottom
2. Wait for confirmation message
3. Changes reflect immediately across the dashboard

## Troubleshooting

### Dashboard won't open?
- Ensure you're logged in
- Refresh the page
- Check your internet connection

### Profile picture upload fails?
- Check file size (must be < 5MB)
- Ensure it's an image file
- Try a different image format
- Check your internet connection

### Charts not displaying?
- Wait for data to load
- Solve some problems first (charts need data)
- Refresh the dashboard

### Changes not saving?
- Check all required fields are filled
- Ensure you clicked "Save Changes"
- Check for error messages
- Verify internet connection
